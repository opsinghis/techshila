## 4. ENGAGEMENT OPTION DETAILS

### 4.1 Standard Engagement (2 weeks)

**Week 1 — Setup & Training**
- Repository assessment and framework installation
- Generator runs tailored to the Client's codebase (Development, Story, and QE pipelines)
- Validation and first end-to-end workflow execution
- 5 training and mindset sessions (approximately 6 hours total, recorded):
  - Vibe Coding to Agentic Delivery (1.5 hrs)
  - The Trust Model (1 hr)
  - Development Pipeline Deep-Dive (1 hr)
  - QE & Story Pipeline Deep-Dive (1.5 hrs)
  - Prompt Engineering for Delivery (1 hr)

**Week 2 — Execution & Handoff**
- Paired workflow execution on real sprint tickets
- Configuration tuning based on team feedback
- Champion coaching and peer session
- Documentation website source code handoff and update walkthrough
- Champion playbook delivery
- Upgrade path documentation
- Engagement retrospective and sign-off

### 4.2 Advisory Add-On (4 weeks post-engagement)

Where the Client has selected the Standard + Advisory option, the following support is included for 4 weeks following the completion of Week 2:

- Up to 2 hours per week of asynchronous support via agreed channels (Slack, email, or Teams)
- Workflow configuration questions
- Champion coaching as they onboard new team members
- Edge case troubleshooting
- Guidance on extending or customising workflows

Advisory support is provided asynchronously. The Supplier shall use reasonable endeavours to respond within 1 Working Day.
